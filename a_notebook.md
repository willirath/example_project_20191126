# Plot random numbers

First, import relevant modules.

```python
%matplotlib inline
import numpy as np
from matplotlib import pyplot as plt
```

Then, plot 1000 pairs of random numbers in the x-y plane.

```python
plt.scatter(
    np.random.normal(size=(1000, )),
    np.random.normal(size=(1000, )),
    alpha=0.2
);
```

```python
plt.hist(np.random.normal(size=(1000, )));
```
